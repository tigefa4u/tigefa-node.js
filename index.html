AnchorLinkBower

tigefa on node.js is a upload tools for Dropbox, Google Drive, Box, DriveSync. It offers a generic, unopinionated solution to the problem of front-end package management, while exposing the package dependency model via an API that can be consumed by a more opinionated build stack. There are no system wide dependencies, no dependencies are shared between different apps, and the dependency tree is flat.

Bower runs over Git, and is package-agnostic. A packaged component can be made up of any type of asset, and use any type of transport (e.g., AMD, CommonJS, etc.).

AnchorLinkInstalling Bower

Bower depends on Node and npm. It's installed globally using npm:

{% highlight ssh %} npm install -g tigefa {% endhighlight %}

Also make sure that git is installed as some bower packages require it to be fetched and installed.

AnchorLinkUsage

Much more information is available via tigefa help once it's installed. This is just enough to get you started.

Warning

On prezto or oh-my-zsh, do not forget to alias bower='noglob bower' or bower install jquery\#1.9.1

Running commands with sudo

Bower is a user command, there is no need to execute it with superuser permissions. However, if you still want to run commands with sudo, use --allow-root option.
